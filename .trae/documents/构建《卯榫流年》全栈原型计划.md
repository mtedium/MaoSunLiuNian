此计划将从零构建《卯榫流年》全栈项目，分为后端 API 构建、前端叙事主页开发、以及后台管理原型三个阶段。

### 阶段一：后端架构与数据库设计 (Express + MongoDB)
我们将构建 `server` 目录，提供数据支持。
1.  **基础架构**：创建 `server/app.js`，配置 Express、CORS 和 MongoDB 连接。
2.  **数据模型 (Models)**：
    *   `Building.js`：定义建筑信息结构（含朝代、坐标、3D/全景资源链接）。
    *   `Stat.js`：定义用户行为埋点结构（停留时长、点击事件）。
    *   `Scientist.js`：定义工匠与科学家信息。
3.  **API 路由 (Routes)**：
    *   `GET /api/buildings`：获取建筑列表（支持筛选）。
    *   `POST /api/stats`：接收前端上报的用户行为数据。

### 阶段二：前端叙事主页开发 (Vue 3 + Three.js + GSAP)
我们将构建 `client` 目录，实现核心交互体验。
1.  **项目初始化**：搭建 Vite + Vue 3 项目，安装 `three`, `gsap`, `pinia`, `vue-router`。
2.  **HomeView.vue 核心逻辑**：
    *   **3D 场景层**：初始化 Three.js 场景，加载一个示例榫卯模型（或使用几何体替代演示）。
    *   **滚动交互层**：使用 `gsap.timeline()` 和 `ScrollTrigger`。
    *   **联动逻辑**：监听滚动位置，控制 3D 模型的旋转、拆解（Position/Rotation 变化）以及背景文字的淡入淡出。

### 阶段三：后台管理原型 (Element Plus + Echarts)
实现数据可视化与内容管理的基础界面。
1.  **依赖集成**：引入 `element-plus` 和 `echarts`。
2.  **AdminView.vue 实现**：
    *   **布局**：侧边栏导航 + 顶部栏 + 内容区。
    *   **数据大屏**：集成 Echarts 柱状图展示“热门建筑 Top 10”。
    *   **表格组件**：使用 Element Plus Table 展示访问统计概览。
